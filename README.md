# Personal Portfolio (Next.js 14 + TypeScript)

This is a minimal, production-ready personal portfolio scaffold using Next.js 14 App Router, TypeScript and Tailwind CSS. Content is sourced from `siteConfig.ts`.

Quick start:

```bash
cd portfolio
npm install
npm run dev
```

Deploy: push to a Git repo and import to Vercel (Next.js 14 supported).
